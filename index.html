常见问题

官网显示 Access Denied 怎么办？
这是 IP 封锁。请更换更干净的节点，或开启无痕模式。如果依然困难，直接使用顶部的国内镜像站是最简单的解决方案。
镜像站数据安全吗？
推荐的镜像站均采用 HTTPS 加密，且通过 API 直接连接 OpenAI。对话数据不会被第三方存储。但请勿在任何 AI 中输入银行卡密码等极端敏感信息。
GPT-5.2 对比 GPT-4o 提升大吗？
对于逻辑推理、复杂代码编写和非英语环境（如中文），提升非常显著。日常闲聊区别不大。
